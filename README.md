# Polarity Productions

A single, static HTML landing page for Polarity Productions—a personal production hub for music, tools, tutorials, and experiments. This site serves as a curated feature overview of the creator's work.

## Overview

This project is designed to be a "digital workshop"—practical, authentic, and creator-driven. It is not a complete archive but a highlight reel of:
- Music production & releases
- Bitwig Studio tutorials & tools
- Presets, samples, and assets
- Experimental web tools
- Written content

## Tech Stack

- **Core**: HTML5, Vanilla CSS, Vanilla JavaScript
- **Frameworks**: None (No React, Vue, Bootstrap, etc.)
- **Rendering**: Static HTML
- **Styling**: CSS Variables for theming, Mobile-First, Dark Mode friendly

## Project Structure

```
/

├── assets/

│   ├── js/
```

## Features

1. **Hero Section**: Introduction and primary links (YouTube, Blog, Dispenser, Bandcamp, Patreon).
2. **Featured Sections**: Grouped by theme (Tutorials, Music, Presets, Tools, Writing).
3. **Minimal Footer**: Essential links.

## Development

This is a static site with no build step.

1. Clone the repository.
2. Open `index.html` in your browser or serve with a simple static server (e.g., `python -m http.server`, `php -S localhost:8000`, or VS Code Live Server).

## Design Guidelines

- **Visuals**: Minimal layout, strong typography, "workshop" feel.
- **Tone**: Personal, calm, confident, practical.
- **Accessibility**: Semantic HTML, ARIA roles, keyboard navigation.
